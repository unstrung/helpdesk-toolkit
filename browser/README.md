# Browser Troubleshooting Script

A PowerShell-based diagnostic tool that collects browser-related connectivity and configuration data. This script helps help desk technicians quickly identify issues with DNS resolution, internet connectivity, proxy configuration, browser cache size, and default browser settings.

---

## 📌 Features

- **[DNS Resolution Tests](ca://s?q=Explain_DNS_resolution_for_browsers)**  
- **[Internet Connectivity Checks](ca://s?q=Explain_internet_connectivity_tests)**  
- **[Proxy Configuration Detection](ca://s?q=Explain_Windows_proxy_settings)**  
- **[Browser Cache Size Analysis](ca://s?q=Explain_browser_cache_and_why_it_matters)**  
- **[Default Browser Identification](ca://s?q=Explain_default_browser_settings_in_Windows)**  

All results are saved to a timestamped report on the Desktop.
