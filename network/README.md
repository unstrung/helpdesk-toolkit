# Network Diagnostic Tool

A PowerShell-based diagnostic script that collects:

- IP configuration
- DNS settings
- Default gateway
- Ping tests
- Traceroute
- Summary section

## Usage

1. Download `Network-Diagnostics.ps1`
2. Right-click → Run with PowerShell
3. A report will be generated on the Desktop:
   `NetworkReport_<COMPUTERNAME>_<timestamp>.txt`

## Purpose

This tool standardizes how help desk technicians gather network information for troubleshooting connectivity issues.

## Example Use Cases

- User reports intermittent internet
- DNS resolution failures
- VPN-related network issues
- Gateway unreachable
