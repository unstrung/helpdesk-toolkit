# Event Log Collector

The Event Log Collector is a PowerShell-based tool that exports key Windows event logs into a timestamped folder on the Desktop. This provides help desk technicians with fast access to the most important diagnostic logs when troubleshooting system crashes, update failures, login issues, or general system instability.

---

## 📌 Features

- **[System Logs](ca://s?q=Tell_me_more_about_System_event_logs)**  
- **[Application Logs](ca://s?q=Tell_me_more_about_Application_event_logs)**  
- **[Security Logs](ca://s?q=Tell_me_more_about_Security_event_logs)**  
- **[Windows Update Logs](ca://s?q=Tell_me_more_about_Windows_Update_logs)**  
- **[Crash / WER Logs](ca://s?q=Tell_me_more_about_Windows_crash_logs)**  
- **[Reliability Monitor Events](ca://s?q=Explain_what_Reliability_Monitor_events_are)**  

All logs are exported in `.evtx` or `.xml` format for easy review or escalation.
