# PC Health Report Tool

A PowerShell-based diagnostic script that collects key system health information to assist with troubleshooting Windows performance issues. This tool is designed for help desk technicians who need fast, standardized system data when responding to user reports such as “my PC is slow,” “apps keep freezing,” or “something feels off.”

---

## 📌 Features

- CPU usage (live performance counter)
- Memory usage (total, used, free)
- Disk space (per volume)
- Key service status:
  - Print Spooler
  - DNS Client
  - Workstation
  - Server
  - Windows Update
- Windows version and build
- Installed updates (Hotfixes)
- Timestamped report saved to Desktop
